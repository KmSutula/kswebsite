YourWeatherNow

Angular - API - TypeScript - HTML - CSS

This website uses location services to determine the weather in your area. Additional locations are also available with user input that connects to the Google Place Autocomplete to fetch suggestions and determine geocoordinates for the requested location. Those geocoordinates are then provided to the OpenWeather API to secure data about the time of day, current temperature, highs and lows, and humidity, as well as render a dynamic icon that visually represents the weather.
